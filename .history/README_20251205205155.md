# Clover Studio Dashboard

Este es el **dashboard de administración** para Clover Studio. Actualmente está en desarrollo, y la versión final permitirá gestionar **productos, colecciones, fragancias y todo lo relacionado con Clover Studio**.

Por ahora, el dashboard **solo permite subir y borrar productos**, mientras se desarrollan las funcionalidades completas del proyecto.

---

## Tecnologías utilizadas

Este proyecto está construido con un stack moderno y tipado, asegurando escalabilidad y facilidad de mantenimiento:

- **Next.js**: Framework principal para el desarrollo fullstack.
- **TypeScript**: Tipado estático en todo el proyecto.
- **Supabase**:
  - Base de datos y almacenamiento.
  - Autenticación de usuarios.
  - Middleware para proteger rutas privadas.
- **Zod**: Validación de configuraciones y formularios.
- **Shadcn/UI**: Componentes UI basados en Radix y Tailwind.
- **TailwindCSS** y **tailwind-merge**: Estilos utilitarios y composición de clases.
- **Radix UI**: Componentes accesibles como Avatar, Dialog, Select, Tooltip, Dropdown, Label y Separator.
- **Cloudinary**: Gestión de imágenes y recursos multimedia.
- **Axios**: Para peticiones HTTP.
- **clsx** y **class-variance-authority**: Gestión de clases CSS dinámicas.
- **Lucide React** y **React Icons**: Iconografía.
- **slugify**: Generación de slugs para productos y colecciones.
- **tw-animate-css**: Animaciones CSS fáciles de usar.

## Scripts disponibles

- `dev`: Levanta el proyecto en modo desarrollo.
- `build`: Compila el proyecto para producción.
- `start`: Inicia la aplicación en modo producción.
- `lint`: Revisa el código con ESLint.
